# MØOD Café ☕✨

A modern café website created for a digital marketing and social media promotion project.  
The project presents the online promotion strategy of a premium coffee and brunch brand through a modern website, visual branding, and social media integration.

---

## 📖 About the Project

MØOD Café is a conceptual premium café brand focused on:
- specialty coffee
- aesthetic brunch experiences
- modern lifestyle branding
- social media marketing
- online customer engagement

The website was designed to simulate the online presence of a real café business and to demonstrate how digital platforms can be used for promotion and brand development.

---

## 🌐 Website Features

- Modern and responsive design
- Premium café branding
- Interactive menu section
- Online reservation form
- Customer reviews section
- Blog-style content section
- Social media integration
- Event promotion section
- Visual gallery presentation
- Contact and location section
